프로젝트 빌드가 완료되면 BUILD SUCCESSFUL in ns 라는 메시지가 뜬다.<br>

build.gradle 빌드에 필요한 옵션들을 정의<br>
build.gradle 에 작성된 빌드 구성을 기반으로 빌드 기능을 수행<br>

우측 gradle 탭을 통해 작업 확인 가능<br>
gradle 탭이 안보인다면 View > Tool Windows > Gradle 클릭<br>
