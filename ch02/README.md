# ⌨ IntelliJ 단축키 모음

* 상단 메뉴바 > Help > Keymap Reference

### 커스텀 단축키 설정하기
* Shift 2번 > keymap 검색
* 단축키 목록 확인 가능
* 단축키 설정하고 싶은 기능을 검색
* 우클릭 > Add Keyboard Shortcut > 사용할 단축키 입력
